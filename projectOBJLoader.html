About The OBJ Loader

The OBJ Loader Parses through any OBJ file and creates a Mesh or Meshes from the Data! It then presents that model in an example area and allows the user to move their camera and lightsource around to see it!

More About OBJ Loader

The OBJ Loader reads through each line in the OBJ file and searches for symbols to give it vertex, index, and Texture data, and then appies it to a newly created Mesh

The L,K,J,and I keys are used to move the light source, and U, and O move the lightsource up and down. The W,S,A, and D keys are used to move the camera, and the E and Q keys rotate the camera.

The OBJ Loader can read from Multiple Meshes and add them all to a single Model, which can be moved and manipulated as a single object
